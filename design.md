# Atelier Community Design System

## Brand Identity
Atelier Community combines the minimalist, architectural aesthetic of Atelier AI with the functional structure of a community platform like Naver Cafe. The design emphasizes whitespace, clear typography, and a "quiet" interface that lets content lead.

## Visual Language

### Color Palette
- **Surface Primary:** `#fbf9f9` (Light, warm grey for main backgrounds)
- **Surface Container:** `#ffffff` (Pure white for cards and content areas)
- **Primary Action:** `#00a862` (Brand green for CTAs, buttons, and active states)
- **On Surface:** `#1a1c1e` (Near black for primary text)
- **On Surface Variant:** `#44474e` (Medium grey for secondary text and labels)
- **Border/Outline:** `#dbdad9` (Soft grey for dividers and structural boundaries)

### Typography (Inter)
- **Headlines:** Bold, high-contrast weights for section headers and post titles.
- **Body:** Regular weight with generous line-height (`1.6`) to ensure readability in long-form community posts.
- **Labels:** Semi-bold, smaller sizes for metadata (dates, view counts, categories).

### Layout & Spacing
- **Grid:** 4px baseline grid for vertical rhythm.
- **Margins:** 16px (Mobile) / 24px (Desktop) consistent outer padding.
- **Rounding:** `ROUND_FOUR` (4px border radius) for a subtle, professional feel.
- **Tonal Layering:** Using background color shifts instead of heavy drop shadows to create depth.

## Component Patterns

### Navigation
- **Mobile:** Bottom navigation bar for core destinations (Home, Search, Profile) and a top app bar with contextual actions.
- **Desktop:** Fixed left sidebar for community navigation (Notices, Popular, Discussions) and a global top bar for user profile and search.

### Content Cards
- Clean borders without heavy shadows.
- Clear metadata hierarchy (Category > Title > Snippet > Author/Date).
- High-quality imagery integrated with rounded corners.

### Interactive Elements
- **Buttons:** Solid brand green for primary actions; outlined or tonal for secondary actions.
- **Input Fields:** Minimalist styling with clear focus states.
- **Comments:** Nested, threaded layout with subtle indentation and clear separation between author and content.

## Design Principles
1. **Fidelity over Decoration:** Prioritize typography hierarchy over decorative elements.
2. **Architectural Layout:** Use intentional whitespace to define boundaries.
3. **Consistency across Viewports:** Maintain the same "quiet" aesthetic from mobile to desktop.
